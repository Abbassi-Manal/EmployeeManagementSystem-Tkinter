
# Guide d'installation

## Étape 1 : Installation des dépendances

Installez les dépendances nécessaires en exécutant la commande suivante dans votre ligne de commande :
					`pip install -r requirements.txt`



## Étape 2 : Exportation de base de données

Exporter base de puis le dossier `Data/employee_management.sql`


## Étape 3 : Lancement de l'application

Exécutez le fichier `App.py` pour lancer l'application 


NB :
    Pour consulter base de données veuiller ouvrir fichier employee_management cituer dans :
					`Data/employee_management.sql`



Amusez-vous ;)
